## Add Cloud Storage Permission

On this demo, we will create/add a cloud storage permission.
<br>

In the Console, select <b>Navigation menu > IAM & Admin > IAM</b>.
<br>
<img width="474" height="425" alt="create-storage-permisssion-1" src="https://github.com/user-attachments/assets/d785eb4d-5e37-44c2-9118-f491009e9f3e" />


Click <b>+Grant access</b> button and paste the Username/email name into the <b>New principals</b> field.
<br>
<img width="323" height="79" alt="create-storage-permisssion-2" src="https://github.com/user-attachments/assets/347b143b-fda1-42f0-903a-b59ba3491e80" />


In the Select a role field, select <b>Cloud Storage > Storage Object Viewer</b> from the drop-down menu.
<br>
<img width="430" height="463" alt="create-storage-permisssion-3" src="https://github.com/user-attachments/assets/db5e7919-dafa-4850-9344-2507a3cc65fe" />

Click <b>Save</b>.
<br>
