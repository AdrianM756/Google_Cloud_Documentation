## Create a Bucket

in this demo, we will create a bucket on google cloud.
<br>

Create a Cloud Storage bucket with a unique name. From the Cloud Console, select <b>Navigation menu > Cloud Storage > Buckets</b>.
<br>
<br>
<img width="401" height="842" alt="create-a-bucket-1" src="https://github.com/user-attachments/assets/0d3feeca-8ca7-4cdd-9e71-72422358c12c" />


Click <b>+Create</b>. We will then name this as ```bucket-test-file```. Click <b>Create</b>.
<br>
<br>
<img width="666" height="157" alt="create-a-bucket-2" src="https://github.com/user-attachments/assets/24463809-6699-439e-878b-d21524c0b224" />

## Upload a sample file
On the Bucket Details page click <b>Upload files</b>.
<br>
<br>
<img width="506" height="190" alt="create-a-bucket-5" src="https://github.com/user-attachments/assets/9939e717-c7db-4981-aa12-f2c3a18650a2" />


Browse your computer to find a file to use. Any text or html file will do. Click on the three dots at the end of the line containing the file and click <b>Rename</b>.
<br>
<br>
<img width="309" height="417" alt="create-a-bucket-6" src="https://github.com/user-attachments/assets/b6213834-c87b-4fd7-9fc7-d1088f646500" />


Rename the file ```sample.txt```. Click <b>Rename</b>.
